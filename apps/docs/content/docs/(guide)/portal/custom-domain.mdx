---
title: Eigene Domain
description: Stellen Sie das Mitgliederportal unter Ihrer eigenen Domain bereit.
---

<Callout type="info">
  Diese Funktion ist im **Pro-Tarif** und höher verfügbar.
</Callout>

Sie können das Mitgliederportal unter einer eigenen Domain bereitstellen, z. B. `portal.ihreschule.de`.

## Einrichtung

1. Öffnen Sie **Einstellungen > Allgemein**.
2. Scrollen Sie zum Abschnitt **Eigene Domain**.
3. Tragen Sie Ihre gewünschte Domain ein.
4. Folgen Sie den DNS-Hinweisen zur Verifizierung.

Nach erfolgreicher Prüfung ist das Portal unter Ihrer Domain erreichbar.
