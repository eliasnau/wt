---
title: Gruppen einrichten
description: Organisieren Sie Mitglieder in Trainingsgruppen.
---

Gruppen bilden in Ihrer Schule z. B. Klassen, Leistungsstufen oder Altersbereiche ab.

## Gruppe erstellen

1. Öffnen Sie im Dashboard den Bereich **Gruppen**.
2. Klicken Sie auf **Neue Gruppe**.
3. Erfassen Sie:
   - **Name**: Anzeigename der Gruppe.
   - **Beschreibung**: Optionaler Zweck der Gruppe.
   - **Standard-Mitgliedsbeitrag**: Optionaler Standardpreis.
4. Klicken Sie auf **Gruppe erstellen**.

## Mitglieder zuweisen

1. Öffnen Sie die Gruppendetails.
2. Klicken Sie auf **Mitglieder zuweisen**.
3. Filtern und wählen Sie passende Mitglieder.
4. Geben sie den Monatlichen Preis an, den das Mitglied Zahlen soll. Dieser überschreibt den Standard-Mitgliedsbeitrag der Gruppe. Wenn sie das Mitglied kostenlos zuweisen möchten, geben sie 0 an.  
5. Bestätigen Sie die Zuweisung.
