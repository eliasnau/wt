The **Payment Batches** section allows you to track all direct debit transactions processed by the system.

## Overview

Navigate to **Finance > Batches** to view a list of all payment batches. Each batch represents a collection of payments collected for a specific billing period.

### Batch Details

The table provides the following information for each batch:

- **Billing Month**: The month for which the payments were collected (e.g., January 2024).
- **Batch Number**: A unique identifier for the batch.
- **Total Amount**: The sum of all transactions in the batch.
- **Transactions**: The total count of individual payments included.
- **Created**: The date the batch was generated.

## Viewing a Batch

To view detailed information about a specific batch:

1. Click the **View** button (eye icon) next to a batch row.
2. Alternatively, click the **More Options** (three dots) menu and select **View Details**.

This will open a sheet showing the individual transactions within that batch.
