Rollen bestimmen, welche Daten Benutzer sehen und ändern dürfen.

Sie verwalten Rollen unter [domain.com/dashboard/settings/members](https://domain.com/dashboard/settings/members).

## Rollen zuweisen

<Steps>
<Step>

### Zur Mitgliederverwaltung gehen

Öffnen Sie **Einstellungen > Mitglieder**.

</Step>
<Step>

### Benutzer auswählen

Suchen Sie den gewünschten Benutzer in der Liste.

</Step>
<Step>

### Rolle ändern

Öffnen Sie das Dropdown **Rolle** in der Tabellenzeile.

</Step>
<Step>

### Speichern

Wählen Sie die neue Rolle. Die Rechte werden sofort aktualisiert.

</Step>
</Steps>

## Standardrollen

| Rolle | Beschreibung |
| :--- | :--- |
| Inhaber | Vollzugriff auf Organisation, Abrechnung und Benutzerverwaltung. Kann nicht von anderen Administratoren entfernt werden. |
| Administrator | Kann Mitglieder, Zahlungen und Benutzer (außer Inhaber) verwalten. Kann die Organisation nicht löschen. |
| Mitglied | Für allgemeine Mitarbeiter mit eingeschränkten Rechten. Kein Zugriff auf sensible Verwaltungs- und Abrechnungsfunktionen. |

## Benutzerdefinierte Rollen

<Callout type="info">
  Diese Funktion ist im **Pro-Tarif** und höher verfügbar.
</Callout>

Erstellen Sie bei Bedarf eigene Rollen mit granularen Berechtigungen.

### Tutorial: Benutzerdefinierte Rolle erstellen

<Steps>
<Step>

### Rollenverwaltung öffnen

Öffnen Sie **Einstellungen > Mitglieder** und wechseln Sie in den Bereich **Rollen**.

</Step>
<Step>

### Neue Rolle anlegen

Klicken Sie auf **Rolle erstellen** und vergeben Sie einen eindeutigen Namen (z. B. `Finanzen (Lesen)` oder `Trainer`).

</Step>
<Step>

### Berechtigungen wählen

Aktivieren Sie nur die Ressourcen und Aktionen, die die Rolle wirklich braucht (Prinzip der minimalen Rechte).

</Step>
<Step>

### Speichern und zuweisen

Speichern Sie die Rolle und weisen Sie sie anschließend Benutzern in der Mitgliederliste zu.

</Step>
</Steps>

## Berechtigungen

Die folgenden Berechtigungen stehen in der Rollenverwaltung zur Verfügung.

### Mitglieder (`member`)

| Berechtigung | Beschreibung |
| :--- | :--- |
| `list` | Mitgliederliste sehen. |
| `view` | Einzelnes Mitglied ansehen. |
| `create` | Neues Mitglied anlegen. |
| `update` | Mitgliedsdaten bearbeiten. |
| `delete` | Mitglied löschen. |
| `export` | Mitgliedsdaten exportieren. |
| `view_payment` | Zahlungsbezogene Mitgliedsdaten einsehen. |

### Finanzen (`finance`)

| Berechtigung | Beschreibung |
| :--- | :--- |
| `view` | Finanzdaten anzeigen. |
| `download` | Finanzdokumente herunterladen. |
| `export` | Finanzdaten exportieren. |

### Statistiken (`statistics`)

| Berechtigung | Beschreibung |
| :--- | :--- |
| `view` | Statistiken ansehen. |
| `export` | Statistikdaten exportieren. |

### Gruppen (`groups`)

| Berechtigung | Beschreibung |
| :--- | :--- |
| `view` | Gruppen anzeigen. |
| `create` | Gruppe erstellen. |
| `update` | Gruppe bearbeiten. |
| `delete` | Gruppe löschen. |
