---
title: SEPA-Konfiguration
description: Konfigurieren Sie die SEPA-Lastschrifteinstellungen für Ihre Organisation.
---

Navigieren Sie zu **Einstellungen > SEPA**, um Ihre Bankdaten und Transaktionsbeschreibungen für SEPA-Lastschriften zu konfigurieren.

## Bankverbindung

Geben Sie Ihre Bankdaten ein, um SEPA-Exporte zu ermöglichen. Alle Felder sind erforderlich, sofern nicht als optional gekennzeichnet.

- **Gläubigername**: Der Name des Gläubigers (Ihrer Organisation), wie er auf dem Bankkonto erscheint.
- **IBAN**: Die internationale Bankkontonummer Ihrer Organisation.
- **BIC**: Der Bank Identifier Code (SWIFT-Code) Ihrer Bank.
- **Gläubiger-ID**: Ihre eindeutige Gläubiger-Identifikationsnummer zum Einzug von SEPA-Lastschriften.
- **Name des Initiators** (Optional): Der Name der Partei, die die Transaktion initiiert, falls abweichend vom Gläubiger.
- **Sammelbuchung**: Aktivieren Sie diese Option, um Transaktionen zu einem einzigen Eintrag auf Ihrem Kontoauszug zusammenzufassen. Dies kann den Bankabgleich vereinfachen.

## Transaktionsdetails

Passen Sie den Verwendungszweck an, der auf den Kontoauszügen Ihrer Mitglieder für verschiedene Transaktionsarten erscheint.

Sie können Vorlagen konfigurieren für:
- **Monatsbeitrag**: Beschreibung für wiederkehrende monatliche Mitgliedsbeiträge.
- **Aufnahmegebühr**: Beschreibung für die einmalige Aufnahmegebühr bei der Registrierung.
- **Jahresbeitrag**: Beschreibung für jährliche Mitgliedsbeiträge.

### Verfügbare Variablen

Sie können die folgenden Platzhalter in Ihren Transaktionsbeschreibungen verwenden, um dynamisch relevante Informationen einzufügen:

| Variable | Beschreibung |
| :--- | :--- |
| `%MONTH%` | Der Monatsname (z. B. Januar) |
| `%YEAR%` | Das Jahr (z. B. 2025) |
| `%MEMBER_NAME%` | Der vollständige Name des Mitglieds |
| `%MEMBER_ID%` | Die eindeutige ID-Nummer des Mitglieds |
| `%JOIN_DATE%` | Das Beitrittsdatum |

**Beispiel:**
Eine Vorlage wie `Monatsbeitrag für %MEMBER_NAME% (%MONTH% %YEAR%)` würde als `Monatsbeitrag für Max Mustermann (Januar 2025)` erscheinen.
