<Callout>
  In diesem Bereich verwalten Sie die **Benutzer der Software** (z. B. Empfang, Trainer, Verwaltung) und nicht die Vereinsmitglieder. Für Mitglieder siehe [Mitglieder verwalten](/docs/school/members).
</Callout>

## Mitarbeitende einladen

1. Öffnen Sie **Einstellungen > Mitglieder**.
2. Klicken Sie auf **Mitglied einladen**.
3. Geben Sie die **E-Mail-Adresse** ein.
4. Wählen Sie eine **Rolle**.
5. Senden Sie die Einladung.

Die eingeladene Person erhält eine E-Mail mit weiteren Schritten.

<Callout type="info">
  Details zu Rollen und Berechtigungen finden Sie unter [Berechtigungen & Rollen](/docs/settings/permissions).
</Callout>

## Benutzerlimits

Die Anzahl einladbarer Benutzer hängt vom Tarif ab.

- **Starter**: Bis zu 2 Benutzer
- **Pro**: Bis zu 5 Benutzer
- **Enterprise**: Unbegrenzt

## Benutzer entfernen

1. Öffnen Sie **Einstellungen > Mitglieder**.
2. Wählen Sie den Benutzer in der Liste.
3. Öffnen Sie das Menü und klicken Sie auf **Mitglied entfernen**.

Dadurch wird der Zugriff auf diese Organisation entzogen.
