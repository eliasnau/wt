Ihr Benutzerkonto kann mehreren Organisationen angehören (z. B. verschiedenen Kampfsportschulen oder Zweigstellen). Auf der Seite **Organisationen** können Sie diese Verbindungen verwalten.

## Organisationen wechseln

![Organization Switcher](/images/account/org-switcher.png)

Wenn Sie Mitglied in mehreren Organisationen sind, können Sie sofort zwischen ihnen wechseln.

1. Klicken Sie oben in der Seitenleiste auf den **Namen der Organisation**.
2. Wählen Sie die Organisation aus der Dropdown-Liste aus, auf die Sie zugreifen möchten.

## Eine Organisation verlassen

Wenn Sie nicht mehr mit einer bestimmten Schule zusammenarbeiten, können Sie die Organisation verlassen.

<Steps>
<Step>
 
### Zu den Kontoeinstellungen gehen

Navigieren Sie zu [Konto > Organisationen](https://domain.com/account/organizations).

</Step>
<Step>
 
### Organisation finden

Suchen Sie die Organisation, die Sie verlassen möchten, in der Liste.

</Step>
<Step>
 
### Verlassen

Klicken Sie auf die Schaltfläche **Verlassen**. Sie verlieren sofort den Zugriff auf alle Daten und Ressourcen innerhalb dieser spezifischen Organisation.

</Step>
</Steps>
