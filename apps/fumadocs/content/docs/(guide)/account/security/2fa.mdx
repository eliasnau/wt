Die Zwei-Faktor-Authentifizierung (2FA) fügt dem Anmeldeprozess einen zweiten Schritt hinzu. Zusätzlich zu Ihrem Passwort werden Sie aufgefordert, einen Code einzugeben, der von einer App auf Ihrem Telefon generiert wurde.

## Was ist 2FA?

2FA erhöht die Sicherheit Ihres Kontos erheblich. Selbst wenn jemand Ihr Passwort errät oder stiehlt, kann er ohne Ihr physisches Gerät (Ihr Telefon) nicht auf Ihr Konto zugreifen.

<Callout type="info">
  Die Aktivierung von 2FA wird **dringend empfohlen** für alle Benutzer, insbesondere für diejenigen mit Zugriff auf sensible Mitgliederdaten.
</Callout>

## 2FA aktivieren

<Steps>
<Step>
 
### Authenticator-App installieren

Stellen Sie sicher, dass Sie eine Authenticator-App auf Ihrem Mobilgerät installiert haben. Beliebte Optionen sind:
- Google Authenticator
- Microsoft Authenticator
- Authy
- 1Password

</Step>
<Step>
 
### Einrichtung starten

Navigieren Sie zu **Konto > Sicherheit** und klicken Sie auf **2FA aktivieren**.

![2FA Setup](/images/account/2fa-setup.png)

</Step>
<Step>
 
### QR-Code scannen

Verwenden Sie Ihre Authenticator-App, um den auf dem Bildschirm angezeigten QR-Code zu scannen.

</Step>
<Step>
 
### Code verifizieren

Geben Sie den 6-stelligen Code ein, der von Ihrer App generiert wurde, um die Einrichtung zu bestätigen.

</Step>
<Step>
 
### Backup-Codes speichern

**Wichtiger Schritt:** Ihnen wird eine Liste mit Backup-Codes angezeigt. Speichern Sie diese an einem sicheren Ort (z. B. in einem Passwort-Manager). Dies ist der *einzige* Weg, um auf Ihr Konto zuzugreifen, wenn Sie Ihr Telefon verlieren.

</Step>
</Steps>

## 2FA deaktivieren

Um 2FA zu deaktivieren, navigieren Sie zu **Konto > Sicherheit**, suchen Sie den Abschnitt 2FA und klicken Sie auf **Deaktivieren**.

<Callout type="warn">
  Wir raten **davon ab**, 2FA zu deaktivieren, es sei denn, Sie wechseln das Gerät oder haben einen bestimmten Grund dazu. Es verringert die Sicherheit Ihres Kontos.
</Callout>
