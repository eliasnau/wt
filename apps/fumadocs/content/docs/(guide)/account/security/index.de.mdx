Die Sicherheit Ihres Kontos ist entscheidend, insbesondere beim Umgang mit Mitgliederdaten. Wir bieten verschiedene Tools, um Ihnen beim Schutz Ihres Kontos zu helfen.

![Security Overview](/images/account/security-overview.png)

Verwalten Sie Ihre Sicherheitseinstellungen unter [domain.com/account/security](https://domain.com/account/security).

<Callout type="info">
  Wenn Sie Fragen oder Bedenken bezüglich Ihrer Kontosicherheit haben, zögern Sie bitte nicht, den **Support zu kontaktieren**.
</Callout>

## Verfügbare Sicherheitsfunktionen

### [Zwei-Faktor-Authentifizierung (2FA)](./security/2fa)
Fügen Sie eine zusätzliche Sicherheitsebene hinzu, indem Sie beim Anmelden einen Code von Ihrem Telefon verlangen.

### [Passkeys](./security/passkeys)
Melden Sie sich sicher mit FaceID, TouchID oder Hardwareschlüsseln an – kein Passwort erforderlich.

### [Sitzungsverwaltung](./security/sessions)
Anzeigen und Steuern, welche Geräte derzeit in Ihrem Konto angemeldet sind.

## Passwortverwaltung

<Callout type="warn">
  **Der Support wird Sie NIEMALS nach Ihrem Passwort fragen.** Wenn Sie jemand nach Ihrem Passwort fragt, handelt es sich um einen Betrugsversuch.
</Callout>

Sie können Ihr Passwort jederzeit auf der Hauptseite für Sicherheit ändern.
1. Geben Sie Ihr aktuelles Passwort ein.
2. Geben Sie Ihr neues starkes Passwort ein und bestätigen Sie es.
3. Klicken Sie auf **Passwort aktualisieren**.
