---
title: Mitglieder verwalten
description: Bearbeiten Sie Mitgliederdaten, weisen Sie Gruppen zu und verwalten Sie Kündigungen.
---

Sobald Sie Mitglieder angelegt haben, können Sie deren Profile verwalten, Informationen aktualisieren und Vertragsänderungen direkt über das Dashboard vornehmen.

## Ein Mitglied bearbeiten

Um die persönlichen oder Kontaktinformationen eines Mitglieds zu aktualisieren:
1. Suchen Sie das Mitglied in der Liste.
2. Klicken Sie auf das Menü **Aktionen** (vertikale Punkte) auf der rechten Seite der Zeile.
3. Wählen Sie **Bearbeiten**.
4. Aktualisieren Sie die erforderlichen Felder im Formular (z. B. Adresse, Telefonnummer, Notizen).
5. Speichern Sie Ihre Änderungen.

## Gruppen verwalten

Sie können Mitglieder bestimmten Trainingsgruppen zuweisen (z. B. "Kinderklasse", "Fortgeschrittene Erwachsene").

1. Öffnen Sie das Profil des Mitglieds (entweder über die Schnellansicht oder die vollständigen Details).
2. Navigieren Sie zum Bereich **Gruppen**.
3. Nutzen Sie die Oberfläche, um das Mitglied Gruppen **hinzuzufügen** oder daraus zu **entfernen**.
4. Änderungen werden sofort wirksam.

## Mitgliedschaft kündigen

Wenn ein Mitglied Ihre Schule verlässt, können Sie dessen Kündigung im System bearbeiten.

1. Suchen Sie das Mitglied in der Liste.
2. Klicken Sie auf das Menü **Aktionen** (vertikale Punkte).
3. Wählen Sie **Mitgliedschaft kündigen**.
4. Ein Dialogfeld erscheint, das das **Enddatum der Vertragslaufzeit** anzeigt.
5. Bestätigen Sie die Kündigung.

Das Mitglied wird als "Gekündigt" markiert, und der Status wird aktualisiert, um das Ende der Vertragslaufzeit widerzuspiegeln.

<Callout type="warn">
Gekündigte Mitglieder werden in der Standard-Listenansicht ausgeblendet. Um sie zu sehen, stellen Sie sicher, dass Sie "Gekündigte Mitglieder einschließen" in den Anzeigeoptionen aktiviert haben.
</Callout>

## Vollständiges Mitgliederprofil

Für einen kompletten Überblick klicken Sie im seitlichen Fenster auf **Anzeigen** und dann auf **Vollständige Details anzeigen**. Die vollständige Profilseite bietet eine umfassende Historie des Mitglieds, einschließlich:
- Komplette persönliche Daten
- Vertragshistorie und finanzielle Details
- Detaillierte Gruppenzuweisungen
- Notizen und Protokolle
